# Массивы в C/C++

## Важное предупреждение

> ⚠️ **Компилятор не следит за границами массива.**  
При работе с массивами программист сам несёт ответственность за выход за их пределы.  
Доступ к элементам вне диапазона приводит к неопределённому поведению (undefined behavior).

---

## Рекомендации по объявлению массивов

### Использование констант для размера массива

Для лучшей читаемости и удобства поддержки кода рекомендуется использовать именованную константу при объявлении массива:

```c
#define SIZE 10
// или
const int SIZE = 10;

int arr[SIZE];

```

Для защиты массива от модификации изпользовать const:
const int day[MONTHS]
Указатели на константы не могут использоватся для изменения значений.

Многомерные массивы
```C
int zippo[4][2];
```

zippo\[0] и zippo = имеют одинаковые значения. начинаются в одной позиции.
zippo\[0] - массив из 2 целых типовю = zippo\[0]\[0]
zippo\[0] - адрес знач с типом int
zippo - адрес обьекта размером 2 значений int

При добавление 1 к указателю или адресу дает значение которое больше исходного на размер указанного обьекта. В этом отношении zippo и zippo\[0] отличаются друг от друга.
zippo + 1 != zippo\[0] + 1
Разименование указателя
\*(zippo\[0]) == zippo\[0]\[0] -> значение типо int.
zippo - это адрес адреса и для получения обычного значения потребуется двухкратное разиминование
Двойная косвеность - адрес адреса или указатель на указатель.
