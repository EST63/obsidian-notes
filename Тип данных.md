# Типы данных в C/C++

## Основные типы данных

| Тип           | Диапазон значений (минимум)                     | Формат в `printf` / `scanf`                     |
|---------------|--------------------------------------------------|-------------------------------------------------|
| `int`         | от -32 767 до 32 767                            | `%d`                                            |
| `short`       | короткое целое (не меньше `int`)                 | `%hd`                                           |
| `long`        | длинное целое (не меньше `int`)                  | `%ld`                                           |
| `unsigned`    | беззнаковое целое                               | `%u`                                            |
| `char`        | символьный тип (обычно 1 байт)                   | `%c`                                            |
| `float`       | с плавающей точкой (одинарная точность)          | `%f`                                            |
| `double`      | с плавающей точкой (двойная точность)            | `%lf` (в `scanf`), `%f` (в `printf`)            |
| `long double` | повышенная точность                             | `%Lf`                                           |

> ⚠️ **Примечание:** Конкретный размер типов зависит от архитектуры и компилятора. Указанные диапазоны — минимальные требования по стандарту.

---

## Стандарты языка и дополнительные типы

### C90 (ANSI C)
- `signed` — явное указание знакового типа (например, `signed int`)
- `void` — тип "ничего"; используется для функций без возврата значения и универсальных указателей (`void*`)

### C99
- `_Bool` — логический тип, хранит значения `0` или `1`. Рекомендуется использовать через заголовок `<stdbool.h>`, который определяет синоним `bool`.
- `_Complex` — тип для комплексных чисел (например, `double _Complex`)
- `_Imaginary` — тип для мнимых чисел (редко используется на практике)

## Типы с плавающей точкой

| Тип             | Диапазон приблизительно         | Точность                   | Примечания                                   |
|------------------|----------------------------------|----------------------------|----------------------------------------------|
| `float`          | \(10^{-37}\) до \(10^{+37}\)     | ~6–7 значащих цифр         | Одинарная точность                           |
| `double`         | значительно шире, чем `float`   | ~15–17 значащих цифр       | Двойная точность; используется по умолчанию для литералов (например, `3.14`) |
| `long double`    | не менее `double`                | не меньше, чем у `double`  | Может быть 80-битным или иметь ещё большую точность в зависимости от платформы |

> ⚠️ При переполнении поведение не определено. Часто результатом является `inf` (infinity) или `nan` (not a number).

## Указатели и адресная арифметика

- `&p` — оператор взятия адреса переменной.
- `*p` — оператор разыменования (получение значения по адресу).
- **Имя массива** — это адрес его первого элемента: `arr` эквивалентно `&arr[0]`.
- Обращение к элементу: `arr[i]` и `*(arr + i)` — по смыслу эквивалентны.
- **Никогда не разыменуйте неинициализированный указатель:**

```c
int *p;
*p = 5; // Опасно! Указатель не инициализирован — неопределённое поведение!
